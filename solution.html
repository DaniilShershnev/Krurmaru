Контрольная работа №1

Уравнения математической физики

Задача 1. Классификация уравнений второго порядка

Условие: Решить уравнение, классифицировать его, привести к каноническому виду и найти общее решение:

(x − y)uxy − ux + uy = 0

Решение:

Шаг 1. Классификация уравнения.

Запишем уравнение в стандартной форме:

Auxx + 2Buxy + Cuyy + Dux + Euy + Fu = G

В нашем случае:

  • A = 0 (нет члена uxx)
  • 2B = x − y, то есть B = (x − y)/2
  • C = 0 (нет члена uyy)
  • D = −1, E = 1, F = 0, G = 0

Вычислим дискриминант:

Δ = B² − AC = [(x−y)/2]² − 0·0 = (x−y)²/4 > 0 при x ≠ y

Вывод: Уравнение гиперболического типа при x ≠ y.

Шаг 2. Нахождение характеристик.

Для уравнения с A = 0 и C = 0 характеристики находятся из уравнения:

A(dy)² − 2B dx dy + C(dx)² = 0
−(x−y) dx dy = 0

Это уравнение выполняется при dx = 0 или dy = 0, что дает два семейства характеристик:

  • ξ = x = const
  • η = y = const

Вывод: Уравнение уже записано в канонических переменных (x, y).

Шаг 3. Нахождение общего решения.

Будем искать решения различного вида.

а) Решения вида u = f(x) + g(y):

Если u = f(x) + g(y), то ux = f'(x), uy = g'(y), uxy = 0.

Подставляя в уравнение: (x−y)·0 − f'(x) + g'(y) = 0, откуда g'(y) = f'(x).

Так как левая часть зависит только от y, а правая только от x, обе части равны константе C:

f'(x) = C, g'(y) = C ⇒ f(x) = Cx + C₁, g(y) = Cy + C₂

Получаем первое семейство решений:

u₁ = C(x + y) + D

Проверка: ux = C, uy = C, uxy = 0

(x−y)·0 − C + C = 0 ✓

б) Решения вида u = F(xy):

Пусть u = F(s), где s = xy. Тогда:

ux = F'(s)·y, uy = F'(s)·x, uxy = F''(s)·xy + F'(s)

Подставляя в уравнение и упрощая:

(x−y)[F''(s)·xy + F'(s)] − F'(s)·y + F'(s)·x = 0
(x−y)[xy·F''(s) + 2F'(s)] = 0

При x ≠ y: s·F''(s) + 2F'(s) = 0

Решая это дифференциальное уравнение:

F''(s)/F'(s) = −2/s ⇒ ln|F'(s)| = −2ln|s| + const
F'(s) = A/s² ⇒ F(s) = −A/s + B = −A/(xy) + B

Получаем второе семейство решений:

u₂ = −A/(xy) + B

Проверка: Подстановка показывает, что решение удовлетворяет уравнению ✓

в) Решения вида u = G(x/y):

Пусть u = G(t), где t = x/y. Аналогичными вычислениями получаем:

(t−1)G''(t) + 2G'(t) = 0

Решая: G'(t) = A/(t−1)² ⇒ G(t) = −A/(t−1) + B

Следовательно:

u = −A/(x/y − 1) + B = −Ay/(x−y) + B

Получаем третье семейство решений:

u₃ = Cy/(x−y) + D

г) Аналогично: u₄ = Kx/(x−y) также является решением.

Общее решение:

Поскольку уравнение линейное и однородное, общее решение представляет собой линейную комбинацию найденных решений:

u(x,y) = A(x + y) + B/(xy) + (Cx + Dy)/(x − y) + E

где A, B, C, D, E — произвольные константы.

Задача 2. Задача Коши для волнового уравнения с источником

Условие: Решить задачу Коши:

utt = 25uxx + sin x
u|t=0 = 1
ut|t=0 = 1

Решение:

Уравнение utt = 25uxx + sin x является неоднородным волновым уравнением со скоростью распространения c = 5.

Шаг 1. Решение однородного уравнения.

Рассмотрим сначала однородное уравнение utt = 25uxx с начальными условиями u|t=0 = φ(x), ut|t=0 = ψ(x).

По формуле Даламбера решение имеет вид:

u₀(x,t) = [φ(x−ct) + φ(x+ct)]/2 + (1/2c)∫x−ctx+ct ψ(s) ds

Шаг 2. Учет неоднородности через интеграл Дюамеля.

Для неоднородного уравнения utt = c²uxx + f(x,t) с нулевыми начальными условиями решение дается интегралом Дюамеля:

uf(x,t) = ∫₀t [(1/2c)∫x−c(t−τ)x+c(t−τ) f(s,τ) ds] dτ

В нашем случае f(x,t) = sin x (не зависит от t).

Шаг 3. Вычисление интеграла от источника.

Внутренний интеграл:

∫x−c(t−τ)x+c(t−τ) sin s ds = [−cos s]x−c(t−τ)x+c(t−τ)
= −cos[x+c(t−τ)] + cos[x−c(t−τ)] = 2sin(x)sin[c(t−τ)]

Поэтому:

uf(x,t) = (sin x)/c ∫₀t sin[c(t−τ)] dτ

Вычисляя интеграл (замена w = c(t−τ)):

uf(x,t) = (sin x)/c² [−cos w]₀ct = (sin x)/c² (1 − cos(ct))

При c = 5:

uf(x,t) = (sin x)/25 (1 − cos(5t))

Шаг 4. Решение с начальными условиями.

Применяем формулу Даламбера для однородного уравнения с нашими начальными условиями φ(x) = 1, ψ(x) = 1:

u₀(x,t) = (1 + 1)/2 + (1/10)∫x−5tx+5t 1 ds = 1 + (1/10)·10t = 1 + t

Шаг 5. Полное решение.

Полное решение задачи Коши:

u(x,t) = 1 + t + (sin x)/25 (1 − cos(5t))

Проверка решения:

Начальные условия:

u(x,0) = 1 + 0 + (sin x)/25·(1 − 1) = 1 ✓
ut = 1 + (sin x)/5·sin(5t) ⇒ ut(x,0) = 1 ✓

Уравнение:

utt = −sin x·cos(5t), 25uxx = −sin x(1 − cos(5t))
utt − 25uxx = −sin x·cos(5t) + sin x − sin x·cos(5t) = sin x ✓

Задача 3. Построение профиля струны

Условие: Построить профиль струны для волнового уравнения:

utt = c²uxx
u|t=0 = φ(x) (по рис. 9)
ut|t=0 = 0

где начальная форма струны φ(x) задана треугольным профилем с высотой h в точке 0 и основанием от −l до l.

Решение:

Шаг 1. Запись начальной функции.

Из графика (рис. 9) начальная форма струны:

φ(x) = { h(1 − |x|/l), если |x| ≤ l; 0, если |x| > l }

Это треугольная функция с вершиной в точке (0, h).

Шаг 2. Применение формулы Даламбера.

Для задачи Коши с ut|t=0 = 0 формула Даламбера имеет вид:

u(x,t) = [φ(x − ct) + φ(x + ct)]/2

Эта формула показывает, что решение представляет собой суперпозицию двух волн:

  • Волна (1/2)φ(x − ct) — движется вправо со скоростью c
  • Волна (1/2)φ(x + ct) — движется влево со скоростью c

Каждая из этих волн имеет амплитуду h/2 (половину от исходной).

Шаг 3. Построение профиля в различные моменты времени.

При t = 0:

u(x,0) = [φ(x) + φ(x)]/2 = φ(x)

Исходный треугольный профиль с вершиной в точке (0, h).

При 0 < t < l/c:

Правая полуволна с центром в ct и левая полуволна с центром в −ct. Обе полуволны имеют высоту h/2.

  • Правая полуволна занимает область [ct − l, ct + l]
  • Левая полуволна занимает область [−ct − l, −ct + l]
  • В области [−ct + l, ct − l] волны перекрываются

При t = l/(4c):

Центры полуволн находятся в точках ±l/4. Высота каждой полуволны h/2.

При t = l/(2c):

Центры полуволн находятся в точках ±l/2. Волны почти разошлись.

При t = l/c:

Две полуволны только касаются в точке x = 0. Каждая имеет высоту h/2.

  • Правая полуволна: центр в x = l, область [0, 2l]
  • Левая полуволна: центр в x = −l, область [−2l, 0]

При t > l/c:

Волны полностью расходятся. Правая волна движется вправо, левая — влево. Между ними образуется плоская область, где u = 0.

Физическая интерпретация:

Начальный треугольный профиль струны распадается на две бегущие волны, каждая амплитудой h/2, движущиеся в противоположных направлениях со скоростью c. Это является следствием принципа суперпозиции для волнового уравнения.

Ответ:

Решение задачи Коши:

u(x,t) = (1/2)[φ(x − ct) + φ(x + ct)]

где φ(x) = h(1 − |x|/l) при |x| ≤ l и φ(x) = 0 при |x| > l.

Профиль струны представляет собой две треугольные полуволны амплитудой h/2 каждая, расходящиеся от центра в противоположных направлениях со скоростью c.

Важные моменты времени:

  • t = 0: начальный профиль φ(x) с высотой h
  • 0 < t < l/c: две полуволны высотой h/2 движутся в разные стороны
  • t = l/c: полуволны полностью разошлись, касаются только в точке x = 0
  • t > l/c: полуволны полностью разделены областью покоя
