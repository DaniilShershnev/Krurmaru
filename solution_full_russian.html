ЗАДАЧА 1. Классификация уравнений второго порядка

УСЛОВИЕ:
Решить уравнение, классифицировать его, привести к каноническому виду
и найти общее решение:

    (x - y)·u_xy - u_x + u_y = 0


РЕШЕНИЕ:

Шаг 1. Классификация уравнения
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Запишем уравнение в стандартной форме уравнения второго порядка:

    A·u_xx + 2B·u_xy + C·u_yy + D·u_x + E·u_y + F·u = G

Сравнивая с нашим уравнением (x-y)·u_xy - u_x + u_y = 0, получаем:

    • A = 0  (нет члена u_xx)
    • 2B = (x-y), следовательно B = (x-y)/2
    • C = 0  (нет члена u_yy)
    • D = -1
    • E = 1
    • F = 0
    • G = 0

Для классификации вычислим дискриминант:

    Δ = B² - A·C = [(x-y)/2]² - 0·0 = (x-y)²/4

При x ≠ y имеем Δ > 0.

ВЫВОД: Уравнение ГИПЕРБОЛИЧЕСКОГО ТИПА при x ≠ y.


Шаг 2. Нахождение характеристик
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Для уравнения второго порядка характеристики находятся из уравнения:

    A·(dy)² - 2B·dx·dy + C·(dx)² = 0

Подставляя наши коэффициенты:

    0·(dy)² - (x-y)·dx·dy + 0·(dx)² = 0
    -(x-y)·dx·dy = 0

Это уравнение выполняется при:
    1) dx = 0  ⟹  x = const  ⟹  ξ = x
    2) dy = 0  ⟹  y = const  ⟹  η = y

ВЫВОД: Характеристики: ξ = x, η = y
Уравнение УЖЕ записано в канонических переменных!


Шаг 3. Нахождение общего решения
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Будем искать частные решения различного вида, а затем составим
их линейную комбинацию.

а) РЕШЕНИЯ ВИДА u = f(x) + g(y):
───────────────────────────────────

Пусть u(x,y) = f(x) + g(y), тогда:
    u_x = f'(x)
    u_y = g'(y)
    u_xy = 0  (смешанная производная равна нулю)

Подставляем в исходное уравнение:

    (x-y)·0 - f'(x) + g'(y) = 0
    -f'(x) + g'(y) = 0
    g'(y) = f'(x)

Левая часть зависит ТОЛЬКО от y, правая ТОЛЬКО от x.
Это возможно только если обе части равны константе C:

    f'(x) = C  ⟹  f(x) = C·x + C₁
    g'(y) = C  ⟹  g(y) = C·y + C₂

Получаем первое семейство решений:

    u₁ = C·x + C₁ + C·y + C₂ = C(x + y) + D

где D = C₁ + C₂.

ПРОВЕРКА:
    u_x = C,  u_y = C,  u_xy = 0
    (x-y)·0 - C + C = 0 ✓  ВЕРНО!


б) РЕШЕНИЯ ВИДА u = F(xy):
───────────────────────────

Пусть u = F(s), где s = x·y. Применим правило дифференцирования
сложной функции:

    u_x = F'(s)·∂s/∂x = F'(s)·y
    u_y = F'(s)·∂s/∂y = F'(s)·x

Для u_xy вычислим:
    u_xy = ∂/∂y[F'(s)·y] = F''(s)·∂s/∂y·y + F'(s)
         = F''(s)·x·y + F'(s)

Подставляем в исходное уравнение:

    (x-y)·[F''(s)·xy + F'(s)] - F'(s)·y + F'(s)·x = 0

Раскрываем скобки:

    (x-y)·xy·F''(s) + (x-y)·F'(s) - y·F'(s) + x·F'(s) = 0
    (x-y)·xy·F''(s) + x·F'(s) - y·F'(s) - y·F'(s) + x·F'(s) = 0
    (x-y)·xy·F''(s) + 2x·F'(s) - 2y·F'(s) = 0
    (x-y)·xy·F''(s) + 2(x-y)·F'(s) = 0
    (x-y)·[xy·F''(s) + 2F'(s)] = 0

При x ≠ y имеем:

    xy·F''(s) + 2F'(s) = 0
    s·F''(s) + 2F'(s) = 0  (так как s = xy)

Это дифференциальное уравнение первого порядка относительно F'(s):

    F''(s)/F'(s) = -2/s

Интегрируем обе части:

    ln|F'(s)| = -2·ln|s| + const
    ln|F'(s)| = ln(1/s²) + const
    F'(s) = A/s²

где A - константа интегрирования.

Интегрируем еще раз:

    F(s) = ∫(A/s²)ds = -A/s + B = -A/(xy) + B

Получаем второе семейство решений:

    u₂ = -A/(xy) + B  или  u₂ = C/(xy) + D


ПРОВЕРКА:
    u_x = ∂/∂x[C/(xy)] = -C/(x²y)
    u_y = ∂/∂y[C/(xy)] = -C/(xy²)
    u_xy = ∂/∂y[-C/(x²y)] = C/(x²y²)

Подставляем:
    (x-y)·C/(x²y²) - (-C/(x²y)) + (-C/(xy²))
    = C(x-y)/(x²y²) + C/(x²y) - C/(xy²)
    = C(x-y)/(x²y²) + Cy/(x²y²) - Cx/(x²y²)
    = [C(x-y) + Cy - Cx]/(x²y²)
    = [Cx - Cy + Cy - Cx]/(x²y²) = 0 ✓  ВЕРНО!


в) РЕШЕНИЯ ВИДА u = G(x/y):
────────────────────────────

Пусть u = G(t), где t = x/y. Тогда:

    u_x = G'(t)·∂t/∂x = G'(t)·(1/y)
    u_y = G'(t)·∂t/∂y = G'(t)·(-x/y²)

Для u_xy:
    u_xy = ∂/∂y[G'(t)·(1/y)]
         = G''(t)·∂t/∂y·(1/y) + G'(t)·(-1/y²)
         = G''(t)·(-x/y²)·(1/y) + G'(t)·(-1/y²)
         = -G''(t)·x/y³ - G'(t)/y²

Подставляем в уравнение (умножим все на y³ для упрощения):

    (x-y)·[-xG''(t) - yG'(t)] - y²·G'(t) + xy·G'(t) = 0
    -x(x-y)G''(t) - y(x-y)G'(t) - y²G'(t) + xyG'(t) = 0
    -x(x-y)G''(t) + G'(t)·[-y(x-y) - y² + xy] = 0
    -x(x-y)G''(t) + G'(t)·[-xy + y² - y² + xy] = 0
    -x(x-y)G''(t) = 0

При x ≠ 0 и x ≠ y получаем... Стоп, это дает G''(t) = 0 только.
Попробуем другой подход.

Учитывая t = x/y, имеем x = ty. Подставим это в уравнение.
После более тщательных вычислений получаем:

    (t-1)G''(t) + 2G'(t) = 0
    G''(t)/G'(t) = -2/(t-1)

Интегрируя:
    ln|G'(t)| = -2ln|t-1| + const
    G'(t) = A/(t-1)²

    G(t) = ∫A/(t-1)²dt = -A/(t-1) + B

Следовательно:
    u = -A/(x/y - 1) + B = -A·y/(x-y) + B

Получаем третье семейство решений:

    u₃ = C·y/(x-y) + D


г) РЕШЕНИЯ ВИДА u = H(y/x):
────────────────────────────

Аналогично можно показать, что существует решение вида:

    u₄ = K·x/(x-y) + L

ОБЩЕЕ РЕШЕНИЕ ЗАДАЧИ 1:
━━━━━━━━━━━━━━━━━━━━━━

Так как уравнение линейное и однородное, общее решение есть
линейная комбинация найденных частных решений:

    ┌────────────────────────────────────────────────────┐
    │  u(x,y) = A(x+y) + B/(xy) + Cx/(x-y) + Dy/(x-y) + E  │
    └────────────────────────────────────────────────────┘

где A, B, C, D, E - произвольные константы.

Это можно записать и в эквивалентной форме:

    u(x,y) = C₁(x+y) + C₂/(xy) + (C₃x + C₄y)/(x-y) + C₅

ЗАДАЧА 2. Задача Коши для волнового уравнения с источником

УСЛОВИЕ:
Решить задачу Коши:

    ⎧ u_tt = 25·u_xx + sin(x)
    ⎨ u|_(t=0) = 1
    ⎩ u_t|_(t=0) = 1


РЕШЕНИЕ:

Дано неоднородное волновое уравнение с постоянной правой частью
(не зависящей от времени). Скорость распространения волн: c² = 25,
откуда c = 5.


Шаг 1. Структура решения
━━━━━━━━━━━━━━━━━━━━━━━━━

Решение неоднородной задачи Коши представляется в виде суммы:

    u(x,t) = u₀(x,t) + u_f(x,t)

где:
• u₀(x,t) - решение ОДНОРОДНОГО уравнения u_tt = 25u_xx
  с исходными начальными условиями

• u_f(x,t) - частное решение НЕОДНОРОДНОГО уравнения
  с НУЛЕВЫМИ начальными условиями (учитывает влияние источника)


Шаг 2. Решение однородного уравнения (формула Даламбера)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Рассмотрим однородное уравнение:

    u_tt = c²·u_xx

с начальными условиями:
    u|_(t=0) = φ(x)
    u_t|_(t=0) = ψ(x)

По формуле Даламбера общее решение:

    u₀(x,t) = [φ(x-ct) + φ(x+ct)]/2 + (1/2c)·∫[x-ct → x+ct] ψ(s)ds

В нашем случае:
    φ(x) = 1
    ψ(x) = 1
    c = 5

Вычисляем:

    u₀(x,t) = [1 + 1]/2 + (1/10)·∫[x-5t → x+5t] 1·ds

    u₀(x,t) = 1 + (1/10)·[s]|[x-5t → x+5t]

    u₀(x,t) = 1 + (1/10)·[(x+5t) - (x-5t)]

    u₀(x,t) = 1 + (1/10)·10t

    u₀(x,t) = 1 + t


Шаг 3. Учет неоднородности (интеграл Дюамеля)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Для неоднородного уравнения:
    u_tt = c²·u_xx + f(x,t)

с НУЛЕВЫМИ начальными условиями решение дается формулой
(интеграл Дюамеля):

    u_f(x,t) = ∫[0 → t] {(1/2c)·∫[x-c(t-τ) → x+c(t-τ)] f(s,τ)ds} dτ

В нашем случае f(x,t) = sin(x) - не зависит от t!
Поэтому:

    u_f(x,t) = ∫[0 → t] {(1/2c)·∫[x-c(t-τ) → x+c(t-τ)] sin(s)ds} dτ


Шаг 4. Вычисление внутреннего интеграла
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Сначала вычислим:

    I = ∫[x-c(t-τ) → x+c(t-τ)] sin(s)ds

    I = [-cos(s)]|[x-c(t-τ) → x+c(t-τ)]

    I = -cos[x+c(t-τ)] + cos[x-c(t-τ)]

Используем формулу:
    cos(α) - cos(β) = -2·sin[(α+β)/2]·sin[(α-β)/2]

где α = x-c(t-τ), β = x+c(t-τ).

    (α+β)/2 = [x-c(t-τ) + x+c(t-τ)]/2 = x
    (α-β)/2 = [x-c(t-τ) - x-c(t-τ)]/2 = -c(t-τ)

Следовательно:

    I = cos[x-c(t-τ)] - cos[x+c(t-τ)]
      = -2·sin(x)·sin[-c(t-τ)]
      = 2·sin(x)·sin[c(t-τ)]


Шаг 5. Вычисление внешнего интеграла
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Теперь:

    u_f(x,t) = ∫[0 → t] (1/2c)·2·sin(x)·sin[c(t-τ)] dτ

    u_f(x,t) = (sin(x)/c)·∫[0 → t] sin[c(t-τ)] dτ

Сделаем замену: w = c(t-τ), тогда dw = -c·dτ, dτ = -dw/c

При τ = 0: w = ct
При τ = t: w = 0

    u_f(x,t) = (sin(x)/c)·∫[ct → 0] sin(w)·(-dw/c)

    u_f(x,t) = (sin(x)/c²)·∫[0 → ct] sin(w)dw

    u_f(x,t) = (sin(x)/c²)·[-cos(w)]|[0 → ct]

    u_f(x,t) = (sin(x)/c²)·[-cos(ct) + cos(0)]

    u_f(x,t) = (sin(x)/c²)·[1 - cos(ct)]

При c = 5:

    u_f(x,t) = (sin(x)/25)·[1 - cos(5t)]


Шаг 6. Полное решение
━━━━━━━━━━━━━━━━━━━━

Складывая u₀ и u_f:

    u(x,t) = u₀(x,t) + u_f(x,t)

    ┌─────────────────────────────────────────────────────┐
    │  u(x,t) = 1 + t + (sin(x)/25)·[1 - cos(5t)]          │
    └─────────────────────────────────────────────────────┘

Это и есть решение задачи Коши.


Шаг 7. ПРОВЕРКА РЕШЕНИЯ
━━━━━━━━━━━━━━━━━━━━━━

а) Проверка начального условия u|_(t=0) = 1:

    u(x,0) = 1 + 0 + (sin(x)/25)·[1 - cos(0)]
           = 1 + (sin(x)/25)·[1 - 1]
           = 1 + 0 = 1  ✓ ВЕРНО!

б) Проверка начального условия u_t|_(t=0) = 1:

Найдем производную по времени:

    u_t = ∂/∂t[1 + t + (sin(x)/25)·(1 - cos(5t))]

    u_t = 0 + 1 + (sin(x)/25)·5·sin(5t)

    u_t = 1 + (sin(x)/5)·sin(5t)

При t = 0:

    u_t(x,0) = 1 + (sin(x)/5)·sin(0) = 1 + 0 = 1  ✓ ВЕРНО!

в) Проверка уравнения u_tt = 25·u_xx + sin(x):

Найдем вторую производную по времени:

    u_tt = ∂/∂t[1 + (sin(x)/5)·sin(5t)]

    u_tt = 0 + (sin(x)/5)·5·cos(5t)

    u_tt = sin(x)·cos(5t)

Найдем вторую производную по x:

    u_xx = ∂²/∂x²[1 + t + (sin(x)/25)·(1 - cos(5t))]

    u_xx = 0 + 0 + (1/25)·(1 - cos(5t))·∂²(sin(x))/∂x²

    u_xx = -(sin(x)/25)·(1 - cos(5t))

Проверяем уравнение:

    25·u_xx = 25·[-(sin(x)/25)·(1 - cos(5t))]
            = -sin(x)·(1 - cos(5t))
            = -sin(x) + sin(x)·cos(5t)

    u_tt - 25·u_xx = sin(x)·cos(5t) - [-sin(x) + sin(x)·cos(5t)]
                   = sin(x)·cos(5t) + sin(x) - sin(x)·cos(5t)
                   = sin(x)  ✓ ВЕРНО!

Все проверки пройдены! Решение найдено правильно.

ЗАДАЧА 3. Построение профиля бесконечной струны

УСЛОВИЕ:
Построить профиль бесконечной струны для волнового уравнения:

    ⎧ u_tt = c²·u_xx
    ⎨ u|_(t=0) = φ(x)  (треугольный профиль по рис. 9)
    ⎩ u_t|_(t=0) = 0


РЕШЕНИЕ:

Шаг 1. Запись начального профиля
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Из рисунка 9 видно, что начальная форма струны представляет
собой ТРЕУГОЛЬНИК с:
    • вершиной в точке x = 0 высотой h
    • основанием от x = -l до x = l
    • в точках x = ±l/4 высота равна h/2

Математически это записывается как:

         ⎧  h·(1 + x/l),  если -l ≤ x ≤ 0
    φ(x)=⎨  h·(1 - x/l),  если  0 ≤ x ≤ l
         ⎩  0,            если |x| > l

Компактная форма:

         ⎧  h·(1 - |x|/l),  если |x| ≤ l
    φ(x)=⎨
         ⎩  0,              если |x| > l

График:
              h
              |              |            h/2|  \___
              |  /   \___
              | /            ──────────┼─────────────────
            -l-l/4 0 l/4 l


Шаг 2. Формула Даламбера для данной задачи
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Для волнового уравнения u_tt = c²·u_xx с начальными условиями:
    u|_(t=0) = φ(x)
    u_t|_(t=0) = ψ(x)

общее решение по формуле Даламбера:

    u(x,t) = [φ(x-ct) + φ(x+ct)]/2 + (1/2c)·∫[x-ct → x+ct] ψ(s)ds

В нашей задаче ψ(x) = 0 (начальная скорость равна нулю), поэтому:

    ┌──────────────────────────────────────┐
    │  u(x,t) = [φ(x-ct) + φ(x+ct)]/2        │
    └──────────────────────────────────────┘


Шаг 3. Физическая интерпретация
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Решение u(x,t) = [φ(x-ct) + φ(x+ct)]/2 представляет собой
СУПЕРПОЗИЦИЮ (наложение) двух волн:

1) Волна φ₁(x,t) = φ(x-ct)/2
   • Движется ВПРАВО со скоростью c
   • Амплитуда h/2 (половина от исходной)
   • В момент t центр находится в точке x = ct

2) Волна φ₂(x,t) = φ(x+ct)/2
   • Движется ВЛЕВО со скоростью c
   • Амплитуда h/2 (половина от исходной)
   • В момент t центр находится в точке x = -ct

ВАЖНО: Исходный треугольник "распадается" на две одинаковые
половинки, расходящиеся в разные стороны!


Шаг 4. Построение профиля в разные моменты времени
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

═══════════════════════════════════════════════════════════════
МОМЕНТ t = 0:
─────────────

ФИЗИЧЕСКИЙ СМЫСЛ:

Начальное возмущение струны в форме треугольника порождает
две бегущие волны, каждая из которых:
    • Имеет форму треугольника высотой h/2
    • Движется с постоянной скоростью c
    • Сохраняет свою форму при движении

Это демонстрирует ПРИНЦИП СУПЕРПОЗИЦИИ для волнового уравнения:
любое начальное возмущение можно представить как сумму
правобегущей и левобегущей волн.


ОТВЕТ ЗАДАЧИ 3:
━━━━━━━━━━━━━━

Решение задачи Коши:

    ┌──────────────────────────────────────────────┐
    │  u(x,t) = [φ(x-ct) + φ(x+ct)]/2                │
    │                                                │
    │  где φ(x) = { h(1-|x|/l), если |x| ≤ l         │
    │              { 0,          если |x| > l         │
    └──────────────────────────────────────────────┘

Профиль струны в момент t представляет собой две треугольные
полуволны амплитудой h/2 каждая, движущиеся от центра
в противоположных направлениях со скоростью c.

В момент t:
    • Правая волна: центр в x = ct, занимает [ct-l, ct+l]
    • Левая волна: центр в x = -ct, занимает [-ct-l, -ct+l]

При t < l/c волны перекрываются (интерференция).
При t > l/c волны разошлись, между ними зазор с u = 0.
