ПРОВЕРКА РЕШЕНИЯ ЗАДАЧИ 3
========================================================

НАЧАЛЬНЫЕ ДАННЫЕ:
-----------------
Начальный профиль струны (треугольник):

φ(x) = h(1 - |x|/l)  для |x| ≤ l
φ(x) = 0             для |x| > l

ПРОВЕРКА ФОРМУЛЫ φ(x):
----------------------
✓ φ(0) = h(1 - 0/l) = h(1 - 0) = h
✓ φ(l/4) = h(1 - (l/4)/l) = h(1 - 1/4) = h·(3/4) = 3h/4
✓ φ(l/2) = h(1 - (l/2)/l) = h(1 - 1/2) = h/2
✓ φ(3l/4) = h(1 - (3l/4)/l) = h(1 - 3/4) = h/4
✓ φ(l) = h(1 - l/l) = h(1 - 1) = 0
✓ φ(-l/4) = h(1 - |-l/4|/l) = h(1 - 1/4) = 3h/4
✓ φ(-l) = h(1 - |-l|/l) = h(1 - 1) = 0

ФОРМУЛА ДАЛАМБЕРА:
------------------
u(x,t) = [φ(x-ct) + φ(x+ct)]/2

Это суперпозиция двух волн:
- Правобегущая: (1/2)φ(x-ct) с центром в x = ct
- Левобегущая: (1/2)φ(x+ct) с центром в x = -ct

ДЕТАЛЬНАЯ ПРОВЕРКА ГРАФИКОВ:
============================

МОМЕНТ t = 0:
-------------
u(x,0) = [φ(x-0) + φ(x+0)]/2 = [φ(x) + φ(x)]/2 = φ(x)

Проверка в ключевых точках:
✓ u(0,0) = φ(0) = h
✓ u(l/4,0) = φ(l/4) = 3h/4
✓ u(l,0) = φ(l) = 0
✓ u(-l,0) = φ(-l) = 0

График: Треугольник от -l до l с вершиной h в точке 0 ✓


МОМЕНТ t = l/(4c):
------------------
Правая волна: центр в ct = l/4
Левая волна: центр в -ct = -l/4

Проверка в точке x = 0:
u(0, l/(4c)) = [φ(0 - l/4) + φ(0 + l/4)]/2
             = [φ(-l/4) + φ(l/4)]/2
             = [3h/4 + 3h/4]/2
             = (3h/4 + 3h/4)/2
             = (6h/4)/2
             = 3h/4 ✓

Проверка в точке x = l/4 (центр правой волны):
u(l/4, l/(4c)) = [φ(l/4 - l/4) + φ(l/4 + l/4)]/2
               = [φ(0) + φ(l/2)]/2
               = [h + h/2]/2
               = 3h/4 ✓

Проверка в точке x = -l/4 (центр левой волны):
u(-l/4, l/(4c)) = [φ(-l/4 - l/4) + φ(-l/4 + l/4)]/2
                = [φ(-l/2) + φ(0)]/2
                = [h/2 + h]/2
                = 3h/4 ✓

Проверка в точке x = l (вершина правой полуволны):
u(l, l/(4c)) = [φ(l - l/4) + φ(l + l/4)]/2
             = [φ(3l/4) + φ(5l/4)]/2
             = [h/4 + 0]/2
             = h/8

Проверка максимумов правой полуволны:
Центр правой волны: ct = l/4
Максимум в точке x = ct = l/4
Высота максимума правой полуволны = (1/2)φ(l/4 - l/4) = (1/2)φ(0) = h/2 ✓

Проверка максимумов левой полуволны:
Центр левой волны: -ct = -l/4
Максимум в точке x = -ct = -l/4
Высота максимума левой полуволны = (1/2)φ(-l/4 + l/4) = (1/2)φ(0) = h/2 ✓

График ДЛЯ t = l/(4c):
- Две полуволны с максимумами h/2 в точках ±l/4
- В центре (x=0) они складываются до 3h/4
- Суммарный профиль имеет два пика по h/2 и провал до 3h/4 в центре

ВНИМАНИЕ! Я допустил ошибку в графике! Давайте пересчитаем правильно:

Правая полуволна (1/2)φ(x-ct):
- При x = ct = l/4: (1/2)φ(0) = h/2 ← МАКСИМУМ
- Область: [ct-l, ct+l] = [-3l/4, 5l/4]

Левая полуволна (1/2)φ(x+ct):
- При x = -ct = -l/4: (1/2)φ(0) = h/2 ← МАКСИМУМ
- Область: [-ct-l, -ct+l] = [-5l/4, 3l/4]

В точке x = 0:
Правая волна дает: (1/2)φ(0-l/4) = (1/2)φ(-l/4) = (1/2)·(3h/4) = 3h/8
Левая волна дает: (1/2)φ(0+l/4) = (1/2)φ(l/4) = (1/2)·(3h/4) = 3h/8
Сумма: 3h/8 + 3h/8 = 6h/8 = 3h/4 ✓

Правильный профиль при t = l/(4c):
- Два максимума высотой h/2 в точках ±l/4
- В центре x = 0 высота 3h/4
- Это трапециевидная форма с двумя пиками


МОМЕНТ t = l/(2c):
------------------
Правая волна: центр в ct = l/2
Левая волна: центр в -ct = -l/2

Проверка в точке x = 0:
u(0, l/(2c)) = [φ(0 - l/2) + φ(0 + l/2)]/2
             = [φ(-l/2) + φ(l/2)]/2
             = [h/2 + h/2]/2
             = h/2 ✓

Проверка в точке x = l/2:
u(l/2, l/(2c)) = [φ(l/2 - l/2) + φ(l/2 + l/2)]/2
               = [φ(0) + φ(l)]/2
               = [h + 0]/2
               = h/2 ✓ (максимум правой полуволны)

График: Два треугольника высотой h/2 с центрами в ±l/2,
        слабо перекрываются около x = 0


МОМЕНТ t = l/c (КРИТИЧЕСКИЙ):
-----------------------------
Правая волна: центр в ct = l, область [0, 2l]
Левая волна: центр в -ct = -l, область [-2l, 0]

Проверка в точке x = 0 (точка касания):
u(0, l/c) = [φ(0 - l) + φ(0 + l)]/2
          = [φ(-l) + φ(l)]/2
          = [0 + 0]/2
          = 0 ✓

Проверка в точке x = l (максимум правой волны):
u(l, l/c) = [φ(l - l) + φ(l + l)]/2
          = [φ(0) + φ(2l)]/2
          = [h + 0]/2
          = h/2 ✓

Проверка в точке x = -l (максимум левой волны):
u(-l, l/c) = [φ(-l - l) + φ(-l + l)]/2
           = [φ(-2l) + φ(0)]/2
           = [0 + h]/2
           = h/2 ✓

График: Две полуволны касаются в точке x = 0,
        где u(0, l/c) = 0 ✓


МОМЕНТ t = 3l/(2c) (ПОСЛЕ РАЗДЕЛЕНИЯ):
--------------------------------------
Правая волна: центр в ct = 3l/2, область [l/2, 5l/2]
Левая волна: центр в -ct = -3l/2, область [-5l/2, -l/2]

Проверка: между волнами зазор (l/2, -l/2) → Нет! Должно быть [-l/2, l/2]

В области [-l/2, l/2]:
- Правая волна: φ(x - 3l/2) = 0 (т.к. |x - 3l/2| > l при x ∈ [-l/2, l/2])
  Действительно: при x = l/2, |l/2 - 3l/2| = |-l| = l → граница!
                при x < l/2, |x - 3l/2| > l ✓

- Левая волна: φ(x + 3l/2) = 0 (т.к. |x + 3l/2| > l при x ∈ [-l/2, l/2])
  Действительно: при x = -l/2, |-l/2 + 3l/2| = l → граница!
                при x > -l/2, |x + 3l/2| > l ✓

Следовательно: u(x, 3l/(2c)) = 0 в области [-l/2, l/2] ✓

График: Две разделенные полуволны с зазором между ними ✓
